# Appendix B: Glossary of Key Terms

| Term | Definition |
|------|-----------|
| Agent | An LLM-powered system that can reason, plan, and take actions autonomously |
| Chain-of-Thought (CoT) | Prompting technique that elicits step-by-step reasoning |
| Checkpoint | Saved state allowing agent resumption after interruption |
| Constitutional AI | Training method using AI feedback guided by explicit principles |
| DPO | Direct Preference Optimization - RLHF alternative without separate reward model |
| Few-shot | Prompting with examples to guide output format/behavior |
| Guardrails | Safety mechanisms that filter/modify agent inputs and outputs |
| Hallucination | Model generating plausible but factually incorrect content |
| Handoff | Transfer of conversation/task from one agent to another |
| HITL | Human-in-the-Loop - involving humans in agent decision workflows |
| LLM-as-Judge | Using an LLM to evaluate another LLM's outputs |
| LoRA | Low-Rank Adaptation - efficient fine-tuning technique |
| MCP | Model Context Protocol - standard for connecting LLMs to external tools |
| RAG | Retrieval-Augmented Generation - grounding LLM outputs in retrieved documents |
| ReAct | Reasoning + Acting pattern for tool-using agents |
| RLHF | Reinforcement Learning from Human Feedback |
| Span | Individual unit of work within a trace (e.g., one LLM call) |
| Supervisor | Orchestrator agent that delegates to specialist agents |
| Swarm | Multi-agent pattern with peer-to-peer handoffs |
| Temperature | Parameter controlling randomness in LLM token selection |
| Tool Calling | LLM capability to invoke external functions/APIs |
| Top-p | Nucleus sampling - selecting from tokens until cumulative probability threshold |
| Trace | Complete record of a request's journey through the system |
